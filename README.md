# Purifier Card

By default, the Home Assistant does not provide any card for controlling air purifiers. This card displays the state and allows to control your air purifier.

![firefox_k9FbYqpqhF](https://github.com/user-attachments/assets/6de5f611-898a-475b-bc38-57a1b283b987)

## Installing
### Manual

1. Go to **HACS**.

<img width="288" height="493" alt="image" src="https://github.com/user-attachments/assets/10240602-c150-4329-95d6-1ec8cb1c259f" />

2. Click on the **three dots** in the upper right corner and select the **Custom repositories** button.

<img width="357" height="455" alt="image" src="https://github.com/user-attachments/assets/219d8f73-16ed-4c62-934b-f2b700033923" />

3. Enter “https://github.com/caglaryalcin/purifier-card” in the **Repository** field. Select Dashboard as the **Type** and click the **Add** button.

<img width="376" height="503" alt="image" src="https://github.com/user-attachments/assets/98bff1e0-000a-407d-99b7-c56b7e4d0b86" />

4. Search for **purifier** card in HACS and download it.

<img width="1916" height="565" alt="image" src="https://github.com/user-attachments/assets/6e5227f8-4276-4b98-8952-c1e1b5123152" />
